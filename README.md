# SpringBoot Vue尿毒症健康管理系统

## 核心功能
- 管理员个人中心
- 患者管理
- 医生管理
- 病史记录管理
- 知识分类管理
- 健康知识管理
- 用药记录管理
- 用药提醒管理
- 透析提醒管理
- 透析结果管理
- 病情预测管理
- 患者交流
- 举报记录管理
- 论坛分类管理
- 系统管理

## 适用场景
- 尿毒症患者的健康管理
- 医疗机构的疾病信息管理
- 医生与患者之间的沟通平台

## 技术特点
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
